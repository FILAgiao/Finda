## 🧠 心理学洞察核心产出（已修正）

### 1. 用户心理画像 ([insights/psychology-map/persona.md](insights/psychology-map/persona.md))

**主动方 (Initiator) 核心心理**:
- **损失厌恶**: 怕"被鸽""遇到骗子" → 诚意金机制
- **决策瘫痪**: 左滑右滑太多 → 每天只推1个人
- **社会身份焦虑**: 怕隐私暴露 → 暗池模式
- **反荡妇机制**: 不能直接表达 → AI中介隐式匹配
- **即时满足**: 想今晚见面 → 即时撮合

**响应方 (Responder) 核心心理**:
- **稀缺性与专属感**: "我只被推荐给1个人"
- **安全感**: 担心骗子/照骗 → 诚意金+身份认证
- **控制感**: 希望有拒绝权

### 2. 用户旅程情绪地图 ([insights/journey-emotion/map.md](insights/journey-emotion/map.md))

**主动方关键发现**:
- ⚠️ **负面峰值**: 等待匹配时的焦虑
- ★ **正面峰值**: 收到"对方接受"确认时的兴奋
- 优化: AI月老进度可视化、匹配成功庆祝动效

**响应方关键发现**:
- ★ **惊喜点**: 收到"有人想请你吃饭"推送
- ⚠️ **选择困难**: 要不要接受 → 每天只推1人简化决策

### 3. 核心差异化功能

| 功能 | 解决的问题 | 心理学原理 |
|------|-----------|-----------|
| **每天只推1个人** | 选择困难 | 决策瘫痪 |
| **诚意金质押** | 被鸽问题 | 损失厌恶 |
| **暗池模式** | 隐私焦虑 | 社会身份焦虑 |
| **AI月老撮合** | 反荡妇机制 | 合理化借口 |
| **即时撮合** | 聊太久不见面 | 即时满足 |

## 🎯 关键修正（v2.0）

### 修正内容

| 错误理解 | 正确理解 |
|---------|---------|
| 服务撮合平台（维修/家政） | 约会/饭搭撮合平台 |
| 需求方 vs 服务方 | 主动方 vs 响应方（双边对等） |
| 服务质量保障 | 防鸽、选择困难、隐私保护 |
| 技能徽章 | 身份认证、诚意金标识 |

### 核心区别

**约会撮合 vs 服务撮合**:
- **交易性质**: 双向选择 vs 单向服务
- **核心痛点**: 被鸽/选择困难/隐私 vs 质量不确定
- **解决方案**: 诚意金/每天1人/暗池 vs 评价系统
