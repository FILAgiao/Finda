# EPIC-001: 用户认证系统

## 目标
建立完整的用户身份体系，支持多端登录和账号安全。

## 范围
✅ 包含:
- 手机号注册/登录
- 微信/Apple ID 第三方登录
- 实名认证流程
- 密码/验证码管理
- Token 刷新机制

❌ 不包含:
- 企业认证 (移至 EPIC-005)
- 多因素认证 (P2 迭代)

## 技术方案
- JWT + Refresh Token 机制
- 短信服务: 阿里云 SMS
- 实名认证: 阿里云金融级实人认证

## 风险
| 风险 | 缓解措施 |
|------|----------|
| 短信轰炸 | 添加图形验证码 + 频次限制 |
| Token 泄露 | 短有效期 + 设备绑定 |
