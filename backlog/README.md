# 需求 Backlog (记忆宫殿的"会议室")

## 组织规则

```
backlog/
├── README.md (本文件)
├── epics/    # 大功能模块
└── stories/  # 具体用户故事
```

## 优先级标记
- 🔴 **P0** - 阻塞发布，必须完成
- 🟡 **P1** - 重要功能，影响体验
- 🟢 **P2** - 优化项，可后续迭代
